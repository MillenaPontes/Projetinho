Sobre mim

Ao longo de algumas semanas, tive a oportunidade de trabalhar com HTML em diversos projetos de estudo. Desde a criação de páginas simples até o desenvolvimento de websites mais complexos, cada experiência uma oportunidade para aprimorar minhas habilidades em Front-end. Foi muito desafiador, pois exigia a criação de um site que fossem atraentes, informativos e fáceis de usar.

Tive que criar um layout responsivo que se adaptasse a diferentes tamanhos de tela, além de utilizar elementos como vídeos, imagens e animações para criar uma experiência de usuário agradável e envolvente. Trabalhar com HTML, CSS e javaScript é algo que me apaixona, pois a cada novo projeto eu aprendo algo novo e desafiador. Tenho certeza que ainda tenho muito a aprender e estou animada para continuar a minha jornada para me tornar uma desenvolvedora Front-end.
